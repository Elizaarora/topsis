To activate this feature:Sign up at EmailJS.Connect an email service and create a template.In index.html, replace the placeholder Public Key, Service ID, and Template ID with your own credentials.üìä How TOPSIS WorksThe algorithm follows a rigorous 6-step mathematical journey:Normalization: Converts different units (like dollars vs. percentages) into a comparable scale.$$r_{ij} = \frac{x_{ij}}{\sqrt{\sum x_{ij}^2}}$$Weighted Normalization: Adjusts the normalized values based on user-defined importance.$$v_{ij} = w_j \cdot r_{ij}$$Ideal Solutions: Identifies the "Best Case" ($V^+$) and "Worst Case" ($V^-$) benchmarks.Distance Calculation: Calculates the Euclidean distance of each alternative from both $V^+$ and $V^-$.Closeness Coefficient: Determines the relative closeness to the ideal solution.$$S_i = \frac{D_i^-}{D_i^+ + D_i^-}$$Final Ranking: Sorts the alternatives from highest score to lowest.üîß Error HandlingBoth tools include robust validation to prevent "garbage in, garbage out" results:File Integrity: Alerts if the file is missing or in an unsupported format.Structure Check: Ensures the file has at least 3 columns.Type Check: Detects non-numeric data in criteria columns.Logic Check: Verifies that the count of weights and impacts matches the number of criteria.Impact Validation: Ensures only + or - characters are used for impacts.üöÄ Quick StartFor the Python Power User:Bashpip install pandas numpy openpyxl
